# 🌐 Multi-Language AI Decision Automation System

**8 Indian Languages × 8 AI Agents × MCP Pipeline**

Enterprise-grade, context-aware AI decision automation supporting **English, Hindi (हिंदी), Telugu (తెలుగు), Tamil (தமிழ்), Malayalam (മലയാളം), Kannada (ಕನ್ನಡ), Odia (ଓଡ଼ିଆ), and Marathi (मराठी)**.

---

## 🏗️ Architecture

```
USER INPUT → LANG_DETECTOR → NLP_AGENT → FEATURE_AGENT → ML_PREDICTOR →
             RULE_AGENT → EXPLAIN_AGENT → ACTION_AGENT → AUDIT_AGENT → OUTPUT
```

### 8 Specialized Agents

| # | Agent | Role |
|---|-------|------|
| 1 | 🔍 **LANG_DETECTOR** | Auto-detect language (langdetect + keyword fallback) |
| 2 | 📝 **NLP_AGENT** | Tokenization, urgency & sentiment extraction |
| 3 | ⚙️ **FEATURE_AGENT** | Feature engineering (context + NLP + language) |
| 4 | 🤖 **ML_PREDICTOR** | Urgency prediction (LogisticRegression ensemble) |
| 5 | 📋 **RULE_AGENT** | Business rules (premium escalation, severity overrides) |
| 6 | 💡 **EXPLAIN_AGENT** | Native-language human-readable explanations |
| 7 | 🎯 **ACTION_AGENT** | Final decision + SLA compliance check |
| 8 | 📊 **AUDIT_AGENT** | SQLite audit trail + compliance logging |

### 8 Language Modules

Each language has its own dedicated module providing urgency keywords, sentiment words, sample texts, and explanation templates — all in the **native script**.

## 🚀 Quick Start

```bash
# 1. Clone the repo
git clone https://github.com/<your-username>/multilang-ai-decision.git
cd multilang-ai-decision

# 2. Install dependencies
pip install -r requirements.txt

# 3. Train the model (generates synthetic data)
python train_model.py

# 4. Launch the dashboard
streamlit run app.py --server.port 8501
```

Then visit **http://localhost:8501** 🎉
